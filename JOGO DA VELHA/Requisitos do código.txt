Classes, Atributos e Métodos:
	1.Classe Tabuleiro // Representa o estado e as operações relacionadas ao tabuleiro.
		- Atributos:
			private char[][] tabuleiro: Matriz 3x3 que representa o estado atual do jogo.
  		- Métodos:
			public Tabuleiro(): Construtor que inicializa o tabuleiro com espaços vazios.
			public void exibirTabuleiro(): Exibe o tabuleiro no console.
			public boolean marcarPosicao(int linha, int coluna, char simbolo): Marca uma posição com o símbolo do jogador, retornando true se for válida e false se não for.
			public boolean verificarVitoria(char simbolo): Verifica se o jogador atual venceu.
			public boolean tabuleiroCompleto(): Verifica se o tabuleiro está cheio, indicando empate.

	2.Classe Jogador // Representa cada jogador com seus dados básicos.
		- Atributos:
			private String nome: Nome do jogador.
			private char simbolo: Símbolo do jogador ('X' ou 'O').
		- Métodos:
			public Jogador(String nome, char simbolo): Construtor que inicializa o jogador com nome e símbolo.
			public String getNome(): Retorna o nome do jogador.
			public char getSimbolo(): Retorna o símbolo do jogador.

	3.Classe JogoDaVelha // Controla a lógica principal do jogo.
		- Atributos:
			private Tabuleiro tabuleiro: Instância da classe Tabuleiro.
			private Jogador jogador1: Representa o primeiro jogador.
			private Jogador jogador2: Representa o segundo jogador.
			private Jogador jogadorAtual: Indica qual jogador está na vez de jogar.
		- Métodos:
			public void iniciarJogo(): Lógica principal que gerencia o fluxo do jogo.
			private void realizarJogada(): Solicita e executa a jogada do jogador atual.
			private void trocarTurno(): Alterna entre os jogadores.
			private void exibirResultado(): Exibe o vencedor ou indica empate.
